搜索命令

以下命令可以帮助你搜索文件或文本：

find 在文件系统中搜索文件或目录

grep 在文本文件中搜索决定的文本模式

locate 基于文件名在数据库中快速找到文件

which 返回可执行文件的路径

ack 特别为程序员设计的一款文件搜索工具。默认会忽略多数版本控制文件夹（如.git, .svn等）的内容

ag（The Silver Searcher）：比ack更快的代码搜索工具，同样默认忽略.git等版本控制文件夹中的内容

whereis 此命令可用于查找二进制程序、源文件、手册页以及其他文件的位置

type 此命令用于确定某个命令是内部shell命令、可执行文件还是别名

apropos 如果你只记得关于某个指令的部分信息，你可以使用的命令描述

alias 如果你经常使用某些linux命令，你可以使用alias命令为这些常用命令创建一个短名，以提升你的工作效率

目录操作命令

mkdir 创建一个新的目录

rmdir 删除一个空目录

tree 以树形结构列出目录内容

du 估计文件和目录的磁盘使用空间

df 显示磁盘使用空间

权限操作命令

在linux系统中，文件和目录的访问可以通过权限操作命令进行控制：

chmod 更改文件或目录的权限

chown 更改为念或目录的所有者和所属的组

chgrp 更改文件或目录的所属组

ping 发送网络请求以测试网络连接

newstat 显示网络连接、路由表等网络状态信息

wget 获取网络资源

进程和系统控制命令

以下是在linux系统中管理进程和控制系统的一些常用命令：

ps 显示当前进程的状态

top 动态显示运行中的进程

kill 发送信号以终止进程

shutdown 关闭机器

reboot 重启机器

logout 退出登录会话

文本操作命令

在编写或处理文本文件时，下列命令可以帮助你完成各种复杂任务：

echo 打印信息到终端

printf 格式化并打印信息

sed 流编辑器，用于对文本文件进行特定的行处理和替换

解释

  • sed 是一个强大的流编辑器，可用于对文本进行替换、删除、插入等操作。
  • s 是 sed 中的替换命令，表示要进行替换操作。
  • /World/Shell/ 是替换规则，World 是要被替换的旧文本，Shell 是用于替换的新文本。
  • g 是全局替换标志，意味着会对输入文本中所有匹配到的 World 都进行替换。不过在这个例子中，由于输入文本是 Hello, world!，而规则里是要替换 World（注意大小写），所以实际上不会发生替换。

awk 在文本文件中进行模式扫描和处理语言

解释

  • echo -e "name\tage\nAlice\t20\nBob\t22"

    ：

    • echo 是用于输出文本的命令。
    • -e 选项的作用是让 echo 命令识别并处理转义字符，像 \t 代表制表符，\n 代表换行符。
    • 双引号内的内容是要输出的文本，最终输出的文本格式如下：
  • |：
    • 这是管道符号，其功能是把前一个命令（echo）的输出当作后一个命令（awk）的输入。
  • awk '{if ($2 >= 21) print $1}'：
    • awk 是一种功能强大的文本处理工具，常用于对文本进行格式化、筛选和分析。
    • {} 内的部分是 awk 的脚本，用于对输入的每一行文本进行处理。
    • if ($2 >= 21) 是一个条件判断语句，$2 代表当前行的第二个字段（以制表符或空格分隔），该语句会判断第二个字段的值是否大于或等于 21。
    • print $1 表示如果条件满足，就打印当前行的第一个字段。

执行过程：

  • echo 命令输出的文本被传递给 awk 命令。

  • awk逐行处理输入文本：

    • 对于第一行 name age，第二个字段 age 无法进行数值比较，所以不满足条件，不做输出。
    • 对于第二行 Alice 20，第二个字段的值是 20，小于 21，不满足条件，不做输出。
    • 对于第三行 Bob 22，第二个字段的值是 22，大于 21，满足条件，会打印第一个字段 Bob。

压缩与解压命令

以下命令主要用于管理和操作文件压缩及解压：

tar 创建、展开及管理tar包

gzip 用于文件压缩或解压

gunzip 用于解压gzip压缩的文件

zip/unzip 创建和解压zip格式的压缩包

磁盘使用管理命令

管理和查看磁盘使用情况：

df 报告文件系统磁盘空间使用情况

du 估计并报告文件及文件夹的磁盘使用情况

fdisk 对磁盘进行分区管理

hdparm 查看或修改 SATA/ATA 磁盘参数

包管理命令

在Debian, Ubuntu及其他基于Debian的系统中，可以使用以下命令进行软件包管理：

apt-get APT包处理工具，用于处理包

dpkg Debian包管理器

在RedHat，CentOS及其他基于RPM的系统中，可以使用以下命令进行软件包管理：

yum 高级软件包管理器，用于处理rpm包

rpm RPM包管理器

进程管理命令

查看和管理正在运行的进程：

ps 报告进程当前状态

top 动态显示当前耗费系统资源最多的进程

htop 比top更友好的动态进程查看工具

kill 终止或发送一个信号到指定进程

pkill 条件地终止或者发送一个信号到指定进程

环境变量命令

查看或设置环境变量：

env 显示当前所有的环境变量

set 显示当前shell所有的环境变量及函数

export 设置或显示环境变量

系统信息发布相关指令

有时，你可能想要查看有关你的系统或硬件的信息。以下命令可以帮助你做到这一点：

uname 会打印操作系统的名称

hostname 打印系统的主机名

dmesg 打印或控制内核环形缓冲区

df 展示文件系统的磁盘空间使用情况

free 展示系统中未使用和已使用的物理和swap内存

uotime 展示系统已经运行了多久，有多少用户正在登录，以及系统负载

last 查看系统的重启和关机记录

w 展示哪些用户正在登录

who 展示哪些用户正在登录，和w命令相似但信息更少

id 展示当前用户的UID，GID以及所在的组

系统控制命令

在一些特殊情况下，你可能需要进行一些系统控制操作。以下命令可以帮助你做到这一点：

halt 关机

reboot 重启系统

shutdown 关机或者重启，和上述两个命令一样，但是提供更多的选项

passwd 更改用户密码

文本编辑器命令

Linux提供了多种命令行文本编辑器。以下这些可能是你需要知道的：

vi/vim vi是一个文本编辑器，而vim是vi的改进版，提供了更多的功能

nano 一个简单易用的命令行文本编辑器

emacs 一个强大的文本编辑器，也是一个定制化的计算环境

其他常用命令

man 查看命令的帮助文档

whatis 显示一个命令的简单描述

whereis 查找命令的二进制文件、源文件及帮助文档

which 查找并显示给定命令的完整路径

whoami 打印当前有效的用户名

date 显示或设置系统日期和时间

cal 显示日历

alias 创建命令别名

unalias 删除别名

history 显示命令历史

clear 清除屏幕或窗口内容

watch 用于实时查看当前命令打印信息
