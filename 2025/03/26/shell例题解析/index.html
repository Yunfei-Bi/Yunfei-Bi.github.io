去掉空行

答案：

解释：

1. !/^$/

这是一个awk的模式匹配条件

  • !是逻辑非运算符，表示取反
  • ^和$是正则表达式中的元字符，^代表行的起始位置，$代表行的结束位置
  • ^$组合起来表示一个空行，即该行没有任何字符
  • 所以!/^$/整体表示匹配那些不是空行的行。也就是说，当输入的行不是空行时，这个条件就会满足。

2. {print $NF}

这是当 !/^$/ 这个条件满足时要执行的动作。

  • {} 是 awk 中用来包裹动作代码块的符号。
  • print 是 awk 内置的打印命令，用于将内容输出到标准输出。
  • $NF 是 awk 中的一个内置变量，NF 代表当前行的字段数量（Number of Fields），$NF 则表示当前行的最后一个字段。

在 awk '!/^$/ {print $NF}' 里，/ 是正则表达式的定界符。在 awk 当中，正则表达式需要被置于两个 / 之间，这样 awk 才能识别出它是一个正则表达式。

打印字母数小于8的单词

描述：

写一个bash脚本以统计一个文本文件nowcoder.txt中字母数小于8的单词。
示例:
假设 nowcoder.txt 内容如下：

你的脚本应当输出：

说明:
不用担心你输出的空格以及换行的问题

答案：

解释：

1. awk -F " "

  • awk 是一个强大的文本处理工具，用于对文本进行格式化和处理。
  • -F " " 是 awk 的一个选项，-F 用于指定字段分隔符，这里将空格作为字段分隔符，意味着 awk 会把每行文本按空格分割成多个字段。

2. {for (i=1; i<NF; i++)

  • {} 是 awk 中的动作代码块，当 awk 读取到每一行时，会执行这个代码块中的内容。
  • for (i=1; i<NF; i++) 是一个 for 循环，i 是循环变量，初始值为 1，表示从第一个字段开始。NF 是 awk 的内置变量，代表当前行的字段数量。循环会遍历从第一个字段到倒数第二个字段（因为条件是 i<NF）。

3. {if (length($i) < 8) {print $i}}

  • if (length($i) < 8) 是一个条件判断语句，length($i) 用于获取第 i 个字段的长度，判断该字段的长度是否小于 8。
  • 如果满足条件（即字段长度小于 8），则执行 print $i，将该字段打印输出。

4. 这里为什么用NF，而不是$NF

NF的含义

NF 是 awk 内置的一个特殊变量，代表当前正在处理的行所包含的字段数量（Number of Fields）。例如，对于一行文本 "apple banana cherry"，按空格分割后，NF 的值就是 3。在 for 循环 for (i=1; i<NF; i++) 中，使用 NF 可以明确循环的终止条件，也就是要遍历从第 1 个字段到倒数第 1 个字段。

$NF的含义

$NF 代表当前行的最后一个字段。在上述 "apple banana cherry" 这个例子里，$NF 就表示 "cherry"。它是对字段的引用，与表示字段数量的 NF 不同。
